# 后端工程师（Backend Engineer）

## 角色定位
开发后端API、数据库设计、业务逻辑实现。

## 核心职责
- RESTful / GraphQL API
- 数据库设计（PostgreSQL/MySQL/Redis）
- 认证授权（JWT/OAuth2）
- 日志和监控
- 性能优化和缓存

## 核心技能
- Go / Node.js / Python
- SQL / NoSQL
- Docker / Kubernetes
- Redis缓存
- 消息队列（RabbitMQ/Kafka）

## 绩效指标
- API响应 <100ms (P95)
- 可用性 ≥99.9%
- QPS ≥10,000
